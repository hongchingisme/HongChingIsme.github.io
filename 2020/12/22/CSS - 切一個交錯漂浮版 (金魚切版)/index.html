linear-gradient 設定漸變效果

linear-gradient（）用於創建一個表示兩種或多種顏色線性漸變的圖片

20deg:設定漸變角度為20,
#a8edea,#fed6e3:第一個顏色到第二個顏色的漸變

漸變可以在這邊找網站，有很多漸變的素材色碼點我

fixed:將漸變固定，這樣上下滑動頁面才不會顯得跑掉怪怪的
center center:這邊是將背景置中 x 與 y軸的概念
100% 100% : 背景大小 這邊是將長寬都設定為100%

這邊是將div內的東西修復排序

align-items:center :將item裡面的內容物水平置中

調整收縮，定位後設定z-index

flex-shrink: 0; :設定為0就會讓flex當中的收縮為0，就不會把版面突出的不分修正
box-sizing: 幫你自動計算扣除 border 或 padding 所剩餘的空間
z-index: 1 : 這裡是設定重疊時誰在上面，數值越大就越上面
z-index可參考:點我

設定margin將圖片重疊，並使用first-child來省事

first-child : 將item下面第一個子元素 進行調整

使用nth-child來對同父異子的元素進行編輯，就不用一個一個設定

nth-child(n) : 這個用法是當設定好的class當中，底下有一樣的元素時，可以個別去編輯，很省事！
