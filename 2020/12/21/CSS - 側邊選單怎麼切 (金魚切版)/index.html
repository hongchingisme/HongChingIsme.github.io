# CSS切版練習

CSS - 側邊選單怎麼切 (金魚切版)

作品

body部分講解

  1. 我們這邊先建立一個 div menu 把整的內容包起來
  2. 輸出一個搜尋與按鈕並用 form 包起來
  3. 接著用 nav 去把我們的連結包起來，因為需要五個連接所以產生五個。
  4. 我們需要 icon 所以在 head 上 輸入: icon 外部套件連結 (等等下面會再說明)然後將選好的 icon 貼進去標籤a裡面。

CSS 部分 (上半部)

  1. 先設定初始值歸零 margin , padding
  2. 將 html 與 body 部分的高度為100%固定
  3. 給背景顏色
  • menu
  1. 設定直立橫幅的寬度與高度(如果這邊沒將 html 的高度設定100%的話，會沒效果，因為你外面沒展開，你的盒子也會展不開)，並且給一個漸層背景 linear-gardient
  2. 將整個menu的 flex 方向從橫向變成直向所以使用 flex-direction: column
  3. 設定一個 padding 值 (隨便調整的)
  4. 因為設定padding後，右邊就跑出拉條，所以輸入box-zizing : border-box 去修復
  5. 設定一個陰影上去比較美觀(自行調整)
  • nav
  1. 給一個padding-top讓他跟上面偏移
  2. 連結a調整，將底線消除，並且 padding讓每個 a有分開
  3. 字體給個黑色，並且排序(可以用 display:flex，也可以像這邊改變為塊元素，這樣就可以有一行的空間，就會排序了，所以也可以用 display:block
  • nav a + a::befor

這個要特別拿出來說，這邊使用::befor是因為要在a標籤之前加入一個偽元素，然後給予一個 border 線，並且定位在父元素a 然後給偽元素寬度後，並且設定 top 調整高度 ，這樣就可以在每個 a 的後面加入一個偽元素的底線了。

後面就是將a裡面的i跟字體寫一個距離

CSS下半部

  • form input
  1. 先排序好表單
  2. 給予一個 border 並且寫入 radius 變成園邊，然後設定左右距離
  3. 在 input 將 border 取消 然後給一個字體，調整高度、寬度距離大小，並給予一個寬度(自行調整)
  4. 把 input 內的背景變成透明的，所以輸入 background-color:transparent就可以透明化
  5. 輸入outline:none可以將輸入時會跑出來的預設框框給關掉
  6. button寫法跟input一樣
  • nav a 動畫

這邊最後要做一個動畫，就是讓滑鼠移入的時候 icon跑出來

  1. 先設定 icon (不同icon可能預設class會不同，要注意) 給予一個margin-right:-1em這樣可以讓文字往圖片靠
  2. 接著輸入transform:scale(0)這邊是 icon的縮放變成0
  3. 設定 hover 滑鼠移動碰到時，margin正常 跟 transform:scale(1) 讓縮放變回來
  4. 最後設定一個transtion讓動畫延遲比較美觀
