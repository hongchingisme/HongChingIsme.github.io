# CSS切版練習

CSS - 超通用橫式版面 (金魚切版)

作品

心得:

該實作並不困難，但老師用到的一些CSS語法有些我沒用過，因為之前用CSS排版都是用float去排，現在使用flex上多了很多用法，所以還在吸收。

語法分析:

  • align-self: flex-end:

align-self的用法是作用在已經套用align-items的屬性。

用法:

  1. flex-start
  2. flex-end
  3. center
  4. stretch
  5. baseline

因為很難用言語去表達他的排序方法，所以上圖片來解釋:

實作語法分析

這邊因為我們想把more標籤移動到右下角，所以使用這個排序方式。

  • flex-direction: column:

設定好的flex的排列方向改變，有四種排列方向:

  1. row：預設值，由左到右，從上到下
  2. row-reverse：與 row 相反
  3. column：從上到下，再由左到右
  4. column-reverse：與 column 相反

實作語法分析:

實作上，因為我們的text這個class裡面有h3 p a 元素，所以當我設定display:flex的時候，他會左到右的去排序(預設是row)，
但我這邊要的排序是從上到下，所以要使用column去把排序改變

  • object-fit: cover:

object-fix 可以修正我們填入的內容物大小與位置。

通常會跟object-position使用

預設為 center center，通常會跟 object-fit 一起使用，定義物件所顯示的 x/y 位移。

幾種用法:

  1. fill
    預設值，預設會強制變形至 css 所定義的元素寬高，不管原檔比例。
  2. contain
    增加或減少載入物件的寬度及高度(維持原比例)直到放得進所定義的元素寬高。
  3. cover
    填滿元素的寬度及高度(維持原比例)，通常會剪掉部分的物件。
    none
    不做任何大小及比例調整。
  4. scale-down
    將會選擇設為 none 或 container 兩者間較小的那個物件。

實作用法分析:

因為這邊我們把圖片放進去之後，會發現圖片比例已經跑掉了，所以使用cover來填滿寬度跟高度並維持原比例。
