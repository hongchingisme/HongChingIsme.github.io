# CSS切版練習

CSS - 對話視窗 (金魚切版)

作品

body

  1. 先用一個 div 包住整個 box
  2. 我們需要一個 div userbox 包住 照片、跟名字
  3. 在 userbox 裡面再包一個訊息文字

CSS 上半部

  1. 一開始先將 body 內置中，並且用 justify-content 將主軸調為中間
  2. 調整 user 內的 排序 利用flex-shrink將縮放關閉，要不然圖片會被縮放變小
  3. 在userbox 內設定好圖片與名字盒子的高度，也調整變置中
  4. 放照片的盒子 pic 給一個圓角
  5. 把 pic 內的 img 給設定為寬度100%去符合容易大小
  6. 設定 vertical-align 將圖片對齊盒子的置中位置
  7. 這邊要在 pic 盒子內加入一個overflow:hidden 將凸出的部分給隱藏起來

CSS下半部

  1. 設定文字訊息 Text 一個 padding 並給圓角，讓畫面優化
  2. 在 remote 的 text 當中左右個要給 margin 因為這邊圖片盒子 pic 是60px 所以要將對話框跟圖片產生距離又不壓縮圖片，所以是要讓margin-left:20 左邊為 margin-right:80px
  3. local的 text部份一樣，但因為 local 的訊息框在右邊，所以要反過來 order:-1 調整 pic 與 text 的順序

繪出一個三角形

這邊要繪出一個三角形並定位在訊息框旁邊，那如何匯出三角形呢？

  1. 設定一個偽元素在 text 前面，並且給他上下右各給邊框，然後將上下使用transparent隱藏，右邊則是顯示出來，接著使用left調整位置就可以了。
  2. 而 local 是反過來的，所以為 right:-10
  3. 最後要在 user 當中加入一個 align-items:flex-start 將版面貼上

心得

這題就有一種切版的感覺出來了，就是一看到版面我大概就在想要怎麼切了，但當然跟老師教得有點差異，但畢竟切版的方法不是只有一種，所以有在持續進步就好。
