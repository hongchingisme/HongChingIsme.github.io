這次學到了一些動畫變化的寫法，很酷，可以好好筆記紀錄一下。

此專案都是將物件定位在 container 裡面，接著利用偏移去定到要的位置，這種寫法我認為不太好，因為如果要做成響應式的話，之後要製作響應式上會很麻煩，等於你又要重切一次。

  • 長度

  • 頻率

  • 角度

  • 時間

  • 數字或者是

  • 整數

  • *：乘。其中一個運算參數必須是數字 (number)
