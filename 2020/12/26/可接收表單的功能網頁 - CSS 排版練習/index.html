心得

此專案學習到很多命名的方式，作者用了很多預設值在 CSS 當中，例如 Margin Padding Font Text 都會設定好規格，這樣在命名的時候直接加上去就可以了，但還是不太習慣，這種用法我應該就有經驗即可，不會當我現在主要的寫法。

接著，作者一定會用一個 container 去包該部分的東西，這樣在做 grid 時比較好，在後面調整縮放也方便。

grid 的用法還要多加了解，向這專案就用切法可以讓左邊大右邊小的切法，值得學習。
