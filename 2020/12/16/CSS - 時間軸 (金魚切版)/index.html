# CSS切版練習

CSS - 時間軸 (金魚切版)

作品

body的部分

這邊我留一個li就好了，其他都是複製的就不多貼了。

這邊用一個box去包住整個時間軸，接著每個小區塊我用card去包起來，在裡面包了標題與內容，這邊因為希望他可以點進去所以用a去包住。

CSS的部分 (如何讓兩邊分開)

這是利用nth-child()的特性將單數雙數隔開，然後加上浮動，一個左一個右，就可以將版面切兩邊，然後使用transformY()將一邊往下移動。

CSS的部分 (如何繪製時間軸)

一開始呢，我們先畫出時間中的中線，先選擇一邊的li給予一個偽元素，並且給一個 border 與高度，接著定位後就慢慢調整到想要的位置。

這邊開始畫入橫線，我們這次選擇a來當偽元素，這邊就不用border，變成給一個寬跟高1px有線條就可以了，一樣要定位在li，然後調整位置就好。

最後要繪製圓型，很簡單，我們變成在h3前面加入偽元素，然後長寬一樣，並給一個border-radius就可以了，一樣要定位，然後慢慢調整位置就可以了！

總結

這次是挑戰自己直接切，所以跟影片教學的會有點不一樣，我剛剛都做完了才發現Amos有加入 hover 但我想應該沒差就沒新增進去了，但主要的寫法感覺差不多，當然Amos的比較簡單一點，但就是自己寫後再看一次影片，就更有印象！
