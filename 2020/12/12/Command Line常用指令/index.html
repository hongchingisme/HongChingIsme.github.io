Command Line常用指令

Command Line的指令中，有幾個比較常用的輸入法:

cd: 進入

當我輸入cd+空格+某個資料夾名稱就會進入該資料夾，如範例:

這邊我是進入一個叫做「git123」的資料夾

那我們想回到剛剛的地方呢?
只要輸入cd ..就可以回到上一頁了!

../.. :上上一頁，上一頁就是 cd .. 以此類推

這邊輸入 cd ..之後就回到F槽了!

ls: 查看

列出當前地方底下的東西有哪些，如範例:

當我輸入ls之後，他告訴我在 git123 資料夾當中有README.md 這個檔案

ls-a:顯示隱藏資料
ls-l:顯示詳細資料

mkdir: 建立資料夾

-
我這邊建立了一個名為 test 的資料夾，建立後可以輸入ls查看當前有哪些檔案

可以發現這邊多了一個 text ,藍色為資料夾的意思

pwd:顯示目前的路徑

當你發現，你忘記你現在在哪個路徑的時候，可以輸入 pwd，它會告訴你目前的路徑在哪裡!就不會迷失方向了!

這邊輸入後它告訴我，我的路徑在F槽裡面的 GIT123 資料夾裡面

touch:建立一個檔案


我這邊輸入了touch index.html，就是要創建一個名為 index 的 html 檔案，創建完畢之後，輸入ls確認

多了一個 index.html 檔案!

rmdir:刪除一個資料夾

我先在這邊創建一個資料夾 apple

輸入 ls 確認 apple 資料夾被創建

之後輸入 rmdir apple來砍掉這個資料夾

使用 ls 來確認檔案已經消失!

刪除完畢!

rm -rf :刪除整個資料夾 要小心使用~!!

這邊就不實測了，寫在這邊是為了提醒自己要小心使用這個語法

cp:複製

前面輸入要複製的檔案名稱，後面為複製出來的檔案名稱，例如:cp index.html cpindex.html

這邊就是我要複製 index.html 然後複製出來的檔案名稱叫做 cpindex.html

接著輸入ls確認，已成功複製

mv:移動

跟複製的邏輯很像，前面是要移動的檔案後面是要移動到哪裡，例如:mv index.html test/

這邊是將 index.html 檔案移動到test這個資料夾裡面

cd 進入 test 資料夾後輸入 ls 確認，檔案成功移動到 test 資料夾裡面

cat:讀取檔案內容

輸入 cat 要讀取的檔案，例如 cat index.html

這裡就成功讀取 index.html 的內文了

clear:清除畫面

這指令可以讓畫面上的指令都消失，當指令過多時可以clear掉複雜的指令，讓畫面更整潔
