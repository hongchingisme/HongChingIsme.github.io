實作一個記帳簿

作品

HTML CSS JS 全部原始碼

本實作想實現的功能:

  1. 送出 form 的時候，將內容提交到 history 如果是 - 的給予minus的class 如果是+的給予 plus 的 class 並新增 li 把這筆資料顯示在 HTML 當中
  2. 讓 Income 的 Value 和 Expense 的 value 相減並寫入 Balance 當中
  3. 按下 history 的 X 按鈕可以刪去 li 要寫入一個 click 事件
  4. 重新整理不會清除紀錄

實作過程遇到的問題

在實作的過程當中，有成功將 li 寫入 HTML 當中，但是計算方面我就搞不定，嘗試了把整個 ul 裡的 li 抓到陣列然後計算，但也會出現錯誤，後來查了一下發現可以透過 reduce 跟 filter 來特別篩選陣列的內容與加總，這邊卡了一段時間。

接著，我抓到了數值之後，我要填入 html 當中一直出現錯誤，當時我的寫法是用 JQuery 當中的 html() 但不知道為什麼都報錯，後來使用回 JS 的 innerHTML 就成功寫入了，我查資料後，好像是當裡面本身沒有值的時候，使用 html() 就會報錯 ，所以在這種情況就要使用 innerHTML

這邊卡了非常久之後，我就跟著影片實作並且理解了。

實作內容

HTML 部分

HTML 的部分，主要是使用一個 container 來包，裡面分別寫入增加與減少金額的 div 與中間的list，最後是寫一個 form 透過提交把內容給推上 list

CSS 部分

在 CSS 當中有兩個寫法比較特殊，所以寫入筆記。

透過實作，發現影片有使用這寫法來分別對不同的 input 來寫入 css ，因為之前沒用過，所以把它紀錄下來。

這方法也是不常使用到，透過 > 來抓取我們要的子元素。

JS 實作過程

首先，我們一樣先抓取 DOM

抓取之後，我們寫入一個測試用的陣列，並且定義變數 transactions 等於這個測試用陣列，而這列當中要放入物件，分別是物件的 ID , text , amount

接著，寫入一個函式，讓我們將測試陣列可以 push 到 list 當中。

在函式中，我們定義一個變數 sign 寫入 if 來判斷物件的 amount 的正負為何，如果是正的給予 + 符號 如果是負的給予 - 符號

接著，我們寫入 item 變數，使用 document.createElement('li') 來新增元素li當我們使用這個 DOM ，也就等於 item = <li></li>

接著我們來判斷這個 li 的正負，若是正的就給予 plus 的 calss 若是負的，就給予 minus 的 class 。

判斷完後，我們將資訊填入 item 當中，使用 innerHTML 可以把內容輸入進去元素當中:

  1. 寫入物件的內容
  2. 寫入 span 元素，並且寫入判斷完成後的 sign 並且寫入物件的數值，前面寫入 Math.abs 是為了讓數值是絕對值，讓數值為正
  3. 寫入一個 button 裡面顯示 x 等等我們會來寫入一個刪除的功能
  4. 將這個 li push 進 list裡面

再來，寫一個初始化表單的功能，讓我們表單為最新的狀況，但只是把 list 清除要怎麼更新呢?
很簡單，把陣列丟進去我們剛剛寫好的函式當中就可以了。

這裡寫入 forEach 讓陣列的每個物件都去執行一次 addTransactionDOM 函式

執行完記得呼叫 itin() 函式

既然完成表單 push 的功能了，我們來寫一個讓金額更新的功能吧！

我們寫入函式　updateValues :

  1. 定義變數　amounts 為 transations 裡面的 .amount ，這裡使用 map 是因為， map 可以將陣列中我們需要的內容，轉換成一個物件返回給我們，也就是說，定義變數後，amounts = transations 當中每個物件的 amount 的陣列
  2. 定義變數 total 為 剛剛取得的 amounts 的 加總，這邊使用 reduce ， 這個用法可以加陣列的數給加總，這邊的 0 是指從陣列的第0個開始加總
  3. 定義變數 income 收入，我們把 amounts 當中 > 0 也就是正的數，使用 filter 抓出來，並且把它加總
  4. 定義變數 expense 收入，我們把 amounts 當中 > 0 也就是正的數，使用 filter 抓出來，並且把它加總
  5. 將收支平衡 (balance) 當中的值為 total
  6. 將收入 (income) money_plus 當中的值為 income
  7. 將支出 (expense) money_minus 當中的值為 expense

寫入提交功能:

  1. 將預設關掉
  2. 寫入 if ，當 input text 裡面的值為空的時候或者 input amount 裡面的值為空的時候，警示 'Please add a text and amount (trim() 是強制讓空白刪除)
  3. 若不是空，有值的話，定義變數 tranction = 物件，裡面有: ID , Text , Amount 將input內容形成物件
  4. 寫入一個函式 generateID 寫入亂數，將這個產生的亂數丟到物件中，給物件一個 ID 好讓我們等等要寫入刪除功能時更方便 (Math.floor()是將裡面的數強制取最近的整數，例如 1.99999 會取 1)
  5. 接著，我們把這個新的物件 push 到 transactions 當中
  6. 丟到 addTransactionDOM 當中判斷該物件正還是負給予 class
  7. 更新金額
  8. 將 input 的內容清空

寫入刪除 item 功能:

  1. 我們使用 filter 來找尋陣列中當中除了該 ID 以外的 ID，並把這個陣列給回 Transaction 這樣 transactions 裡面就只有除了這個物件以外的物件
  2. 執行初始化功能，讓表單刷新
  1. 在 item 的 內容記得要加上一個點擊事件把寫入的函式帶入

寫入 LocalStorage 功能:

  1. 刪除原本的寫入的測試陣列 dummyTransactions
  2. 我們將 transactions 存入 JSON 裡面
  3. 定義變數 localStorageTransactions 去讀取剛剛存入的 JSON
  4. 定義變數 transactions 為讀取下來的陣列，判斷他是不是空的，如果是空的話給予 localStorageTransactions 陣列，接著瀏覽器就會再去執行我們上面的函式，若不是為空的話，就是[]一開始的空陣列。
