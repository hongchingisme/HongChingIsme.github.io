陣列

陣列的定義方法

可以透過[]定義其陣列內容，很像把資訊放在一個盒子裡面的概念，以下是陣列定義的詳細方法:

  • var arr1 = []; //定義一個陣列，但為空的陣列。

  • var arr2 = [1,2,3,4]; //陣列中有四個數據，透過,將數據分開

  • var arr3 = [,,] // 三個空陣列 (但要注意，瀏覽器會判斷為2個，例如IE、google)

  • var arr4 = new array(); //創建空陣列

  • var arr5 = new array(5); //創建空陣列 但裡面有五個空陣列

  • var arr6 = new array(1,2,3,4); // 如同 [1,2,3,4]
