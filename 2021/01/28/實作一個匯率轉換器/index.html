  1. 透過觀看成品，有串接一個匯率的 API，透過API可以抓取我們要的貨幣最新的資料
  2. 寫一個 input ，可以往上下 加減數字，讓上面數字呈上匯率顯示在下面的輸出
  3. 要寫入一個滑鼠案件來讓兩個貨幣位置交換 並更新匯率

實際上碰到的問題

一開始接觸 ajax

因為從來沒有串接過 ajax 所以在一開始就卡住了，後來看了一些文章也都似懂非懂，最後在這篇文章看到方法，解釋的蠻清楚的，但問題來了，我已經串接到了我要的api，但我不知道要怎麼抓，後來嘗試好多次，最後放棄自己思考，去看其他同學的原始碼。

成功抓到我們要的資料

後來發現是因為，在 ajax 的連結中，我要加入自己定義好的變數時，要使用``來包覆住連結才可以在後面加入我的變數

計算問題

一開始我用成功將抓出來的幣值去放在進去 rate 裡面，但問題是，我不知道怎麼去計算，後來發現不是我不會計算，是我在:

當中，我寫成(AmountOne * rate) 而不是 ((AmountOne * rate))，在運算中要再用一個括號，這邊就是基礎不穩，下次要再變數當中計算要記得誇號起來，最後計算有成功把值顯示再計算機當中。

顯示文字上的問題

這邊我想好久都寫不出來，因為我知道可以透過 .text() 去改內容，但我寫的方式都錯誤，後來看其他同學才知道，我要把變數內容寫進去除了要使用``以外，還要使{}來包住變數才可以

結語

這個專案做完後，其實 code 沒幾行，但是我大概花了五個多小時一直再反覆測試要怎麼寫，好的方向是，至少我在實作之前自己思考專案要怎麼進行，而實際上也差不多，但不好的方向是，我大概只是想出方向，在寫法上都會有錯誤導致失敗，最後還是看別人怎麼寫來修改自己的內容，可能就是要多寫一點，用法跟寫法才會記起來！
