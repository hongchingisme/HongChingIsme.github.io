js 分析

首先，我們使用 js 抓取 DOM

接著寫入錯誤訊息的函式，在函式裡面宣告 formControl 為 input.parentElement; 這邊這個 parentElement 的意思是指父元素，例如我們這邊帶入 username ， username 在 html為 input 的 ID ，所以會抓取到 input 的父元素，也就是 div

抓取到 div 後，寫入 form-control error 為 div 的 className 記得 N 要大寫哦 ，然後寫入 querySelecto直接抓取 smaill標籤 ，讓標籤內文為欲輸入的內文。

接著寫入事件選擇器，因為我們要做到當使用者輸入完畢後，就觸發的事件，所以使用 keyup ，然後將函式的預設為空，因為基本的 js 呼叫都有預設值，例如alert就會預設跳出一個彈跳視窗，這邊就是利用preventDefault將預設調整為空取消預設功能。

接著寫入循環

如果內文 value 值 為 0 ，要跳出 xxxxxx is required

email這邊要注意，因為我們要寫入驗證信箱的功能，所以要先寫入正則表達式

這邊寫入正則後，寫入String(email).toLowerCase()讓字串裡面的英文字母變成小寫，接著將 email 的循環加入這個函式

else if (!isValidEmail(email.value)){ showError(email,'Email is not valid');

最後寫入兩個函式，一個為密碼長度設定，另一個式確定第二次輸入密碼是否正確

記得要在呼叫函式在 keyup 事件當中
