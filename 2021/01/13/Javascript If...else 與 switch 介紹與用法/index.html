if的基本寫法

當 if 裡面的結果為 ture 時，警告視窗跳出字串 “Hello”

if(false){
alert(“Hello”)
}

當 if 裡面的結果為 false 時則就不會跳出警告視窗

如果與否則的if寫法

if(5>3){
alert(“yes”);
}else{
alert(“no”)
}

如果 5>3 的話，警告視窗跳出 “yes” 若不是則跳出 “no”
