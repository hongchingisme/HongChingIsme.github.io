4.算術運算子

基本上就是 + - * / % 的觀念

例如：

1 + 2

1跟２為運算元

＋　為運算符

整體　1+2 可以被稱為運算式

在加減乘除當中，有一個很重要的運法就是 %

也就是取餘數，例如:

10/4 = 2…..5

如果是這樣寫 10%4 結果就會是 5
