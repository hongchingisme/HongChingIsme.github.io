# Javascript 作品

實作一個用聲音猜數字的遊戲

作品

原始碼(有原始 JS 的 code)

本實作想實現的功能:

  1. 抓取使用者的聲音轉成文字
  2. 判斷是否為數字
  3. 判斷數字是否超過範圍
  4. 判斷是否符合隨機產生的數字
  5. 循環語音辨識

實作內容

HTML 部分

這次 HTML 部分不多，所以就直接貼上來。

  1. 一開始先插入麥克風的圖片，之後輸入標題。
  2. 寫入一個 msg 包住訊息
  • 成品:

  • 可以透過上方連結觀看詳細程式碼

CSS 部分

CSS 部分沒有特別的地方，唯一要注意的就是在對 span 操作大小時要注意 inline 要改成 inline-block ，因為 span 是行內元素，要把它改成行內區塊元素。

  • 成品:

Jquery 部分

抓取 DOM 與產生亂數

  1. 抓取 DOM
  2. 寫入亂數Math.floor(Math.random()*100) +1這邊+1是因為要讓亂數為99時或0時，變成100跟1
  • 可以透過上方連結觀看詳細程式碼

寫入語音辨識 API

JS 針對語音辨識有相對應的 API 但是在一些瀏覽器上會有不支援的問題。
(必須是在網路上或者自己架一個localhost才可以使用，一開始沒有架設localhost，會導致一直詢問是否可以使用麥克風)

  1. 寫入 SpeechRecognition 語音辨識 API
  2. 寫入 webkitSpeechRecognition 相容瀏覽器
  3. 定義變數 recognition 為一個新的 語音辨識 (上兩部是要讓先讓瀏覽器抓取)
  4. recognition.start() 讓語音辨識開始

寫入語音辨識監聽與返還語音判斷

  1. 寫入事件監聽事件，要使用 result 才會返還語音辨識的結果
  2. 抓取語音辨識的位置與值 e.results[0][0].transcript;


可以看見語音辨識的位置在返還值裡面的 results 陣列中的陣列裡面，所以才要寫入 [0][0]

  1. 寫入函式writeMessage(msg);將辨識的訊息寫進 DOM
  2. 寫入函式 checkNumber(msg)

checkNumber(msg)函式：

  1. 先定義變數 num 為 +msg 這裡的 + 是要讓他為數字

  2. 寫入 if 並且寫入 Number.isNaN(),判斷是否為數字，若不是數字就會為 true 就會顯示訊息

  3. 接著寫入判斷數字是否超過範圍的 if

  4. 寫入 if 判斷如果等於隨機產生的亂數時，顯示訊息

  1. 寫入指引 if ，引導玩家可以更接近答案

事件結束與循環

  1. 寫入監聽事件 end 並且當觸發函式，該函式為讓語音辨識啟動，達到循環
  2. 寫入獲勝時的按鈕功能，讓頁面重新整理重新啟動遊戲
