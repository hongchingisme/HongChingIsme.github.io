本實作想實現的功能:

一個會自動更新年份的新年倒數計時器

實作內容

HTML 部分

HTML部分沒有太複雜的寫法，基本上寫入一個包住時間的 container ，然後底下加入一張讀取的 gif 標籤，之後會在 js 當中來做設置

CSS 部分

CSS 部分比較特別的地方就是覆蓋一個偽元素在 body 當中 ，有一個漸層的效果

覆蓋前:

覆蓋後:

記得覆蓋後有圖層的問題，所以要對 body 進行圖層設置

接著我們將倒數的 conatiner 給 diplay:none; 等等會再 JS 進行操作

Jquery 部分

抓取 DOM 與使用 JS API 抓取當今年份與明年年份

  1. 定義變數 currentYear 並使用 JS 內建的函式 當中的 Date()抓取目前的日期與時間
  2. 抓取後寫入 getFullYear 可以獲取當中的年份
  3. 寫入 newYearTime 來獲得明年 一月一號的時間點

寫入倒數時間的函式

  1. 寫入時間倒數函式
  2. 設定一個新的變數 currentTime為當前時間
  3. 定義變數 diff 將 明年一月一月號的時間點newYearTime 減去 currentTime 會得到離明年差多少的時間(會返還毫秒)
  4. 定義天數的變數，記得要使用 Math.floor()才可以把數字取正，接著除1000得到秒，再除60得到秒，再除60得到分再除24就會得到天數。
  5. 定義小時的變數，除1000得到秒，除60得到分，除60得到小時，後面要寫入%(商)取得少於24小時的數，可得到小時
  6. 後面的分與秒一樣的寫法
  7. 將得到的數值寫入 DOM

寫入 setInterval 讓函式一直執行一直更新

這裡是原生函式用法，後面記得寫入毫秒。

更新下一年的年份進入 DOM 與加入讀取 GIF

  1. 將下一年寫入 DOM

  2. 寫入 setTimeout 函式，因為我們重整會發現網頁需要讀取一下，透過這個 gif 可以增加網站美觀

  3. 將 loading remove 掉

  4. 將 countdown 的 container display:flex 將它顯示出來

  5. 記得要寫入毫秒
