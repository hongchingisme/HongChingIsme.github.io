實作一個唱盤會旋轉的音樂播放器

作品

Html , CSS , Jquery 原始碼

本實作想實現的功能:

  1. 可以播放音樂
  2. 可以跳到下一首跟上一首
  3. 點擊時播放 icon 會變
  4. 顯示音樂播放到哪時改變 width 達成讀取條的效果
  5. 選擇讀取條的位置可以跳轉到讀取條的音樂時間軸

實作過程遇到的問題

一開始我沒看影片,有成功將 icon 轉換，但後來不知道如何去讓音樂播放，自己 google 有 google 到辦法，也有成功讓音樂播放，但接著就是要如何讓歌曲可以隨著音樂時間，讓上面的進度條跟著動我就卡住了，所以最後還是看教學影片去理解。

教學影片是 javasricp 後來我自己再寫一次專案，然後是用 JQuery 來寫，有碰到幾個問題

  • 第一個問題:

不能像 js 那樣直接更改 src 裡面的內容

後來成功 google 到答案，要透過 attr 去抓取該 id 的屬性才可以調整

  • 第二個問題:

我要讓歌曲播放時，發現不能播放

後來也有成功解決問題，因為 JQuery 當中要使用 get()來獲取 DOM 元素所以要這樣寫:

因為 audio.play() 的用法是原生 js 的　api　，所以不可以在　jQuery 直接這樣寫，要使用 get 去抓取 DOM 才可以

Play/pause HTML 5 video using JQuery

抓取 audio 當中的 duration 跟 currentTime 出現問題

我本來直接透過 audio.duration 跟 currentTime 來抓，但都會抓不到我要的值，後來發現有人有在網路上問過跟我一樣的問題，後來是使用 this.來解決的

然後在事件當中，如果要直接抓取 duration 跟 currenTime 但並不是在 audio 事件當中時，要透過 audio[0].currentTime 這種方式才拿的到值，要不然會顯示 NAN ，因為要設定 [0] 才可以找到 index 裡面的 audio DOM element

然後接著又遇到一樣的問題，我在寫下一個點擊的事件也是，直接使用 click 無法執行，要在前面加入 bind 才可以，我 google 後的理解是，寫入 bind 事件，才可以在事件內抓取 audio 的 api

Jquery 部分

一開始我們先抓取 DOM

將歌曲資訊輸入進去 html 當中

  1. 接著我們寫入一個歌曲的陣列，陣列當中要使用歌曲的名字，這樣等等再寫函式的時候可以利用抓取陣列字串把他丟進 HTML 裡面顯示
  2. 設定變數 songsIndex = 1 讓陣列當中第二個數據變成預設值
  3. 呼叫函式 loadsong(songs [songsIndex]); 參數為歌曲陣列，然後是剛剛定義好的變數，這樣可以讓 loadsong 去判斷是陣列中哪個數據
  4. 寫入函式 loadsong(song)
  5. 讓 title 寫入 (song) 就可以將當前陣列的數據寫入
  6. 設定 audio ， 使用 attr 抓取 src 屬性，並且將路徑更改為陣列名稱的 mp3
  7. 設定 cover ， 使用 attr 抓取 src 屬性，並且將路徑更改為陣列名稱的 mp3

這樣就完成了，可以成功將當前的歌曲名稱、歌曲、圖片輸入進去 html 當中！讚讚讚

寫入點擊事件，可以更換 icon

  1. 寫入點擊事件
  2. 事件當中，我們寫入一個 if 判斷，當musicContainer當中，有 play 的 class 時，會執行 pauseSong 函式，若沒有，則執行 playSong 函式
  3. 寫入 playSong 函式:
    點擊的時候，增加 play class 接著將 playBtn 裡面的 i.fas class 刪除 fa-play 的 icon，並且加入 fa-pause 的 icon ，讓我們點播放的時候可以讓 icon 轉換成暫停，增加互動
  4. 寫入 pauseSong 函式:
    移除 play class 並且增加 fa-play 刪除 fa-pause 原理跟上面一樣
  5. 使用 audio ，因為用 JQuery 來寫所以在使用 audio 的 js api 的時候，要先用 get() 來取得陣列當中的初始值才可以執行

寫入 下一首 與 上一首功能

接著寫入點擊事件:

  1. 下一首歌曲的點擊事件

  2. 寫入 songsIndex++ 讓使用者點及下一首歌曲的 icon 時，讓陣列自動+1選取下一個數據

  3. 寫入判斷，如果songsIndex 大於 歌曲陣列 -1 的長度時執行，意思就是，當我們的 ++ 大於我們陣列的數據數量，我們就讓它執行 songsIndex = 0 讓陣列回到一開始的狀態

  4. 載入歌曲

  5. 播放歌曲

  6. 上一首歌曲的點擊事件

  7. 寫入 songsIndex-- 讓使用者點及下一首歌曲的 icon 時，讓陣列自動-1選取上一個數據

  8. 寫入判斷，如果songsIndex 小於 0 ，我們就讓它執行 songsIndex = s.length - 1 讓陣列變成最後一首歌曲

  9. 載入歌曲

  10. 播放歌曲

讓歌曲進行時，讀取條的顏色更著改變

  1. 寫入 timeupdate 監聽事件，要使用 on 要不然會抓不到 timeupdate ，因為這是 js 的 audio api 事件用法
  2. 設定變數為當前的 duration (duration 為 js audio 當中 api 的用法，用來抓取音頻的長度
  3. 設定變數抓取當前的音頻位置 (屬性會返還音頻當前位置 (秒針表示) 當設定該屬性時，播放會跳到指定位置
  4. 設定 progressPercent 變數，讓數值為 音頻位置/音頻長度 *100 這樣就可以抓到當前音樂的位置
  5. 將數值導入 progress 的 width 當中，這樣就時間可以一邊撥放一邊更新 bar 的功能

寫入點擊 bar 時，直接跳轉到當前點擊的位置

  1. 設定變數 width = progressContainer 的寬度
  2. 設定變數 clickX = 抓取 progressContainer X 座標，可以得到 progressContainer 與頁面的距離
  3. 定義 duration 為 音頻的長度 (要注意這邊要使用 [0] 來抓到 DOM ，因為 JQuery 不像 JS 這樣可以直接使用 audio api ，要先透過抓取到陣列才抓到陣列的數據的 DOM)
  4. 設定變數page 為當前點擊的位置與頁面的距離
  5. 將音頻的位置 (目前點擊的距離 - progressContainer 距離 ) / 總寬度 *音頻長度
