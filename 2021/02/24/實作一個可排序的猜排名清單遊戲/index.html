實作可排序的清單

原始碼(有原始 JS 的 code)

本實作想實現的功能:

  • 完成一個可以排序的清單
  • 清單同時有交換與確認清單位置是否正確的功能
  • 練習如何使用 JS API 去完成滑鼠拖曳事件

實作內容

HTML 部分

HTML部分不太多，所以就直接貼上來講解。

  1. 寫入標題
  2. 寫入 <p>段落引導使用者
  3. 寫入表單 <ul> 等等會寫入 JS 來填入 <ls>
  4. 寫入送出按鈕，會再 JS 寫入事件，來判斷表單內容順序

CSS 部分

CSS 部分我個人認為沒有比較特殊的用法，所以就不特別講解了，想看詳細程式碼可以透過上面連結觀看。(主要就是排版置中)

完成後(中間清單會再等等的JS中加入):

Javascript 部分

抓取 DOM 與設定要的陣列與空陣列

  1. 首先我們先抓取 DOM
  2. 接著寫入我們要的固定陣列，數據順序要是設定好的，這樣之後寫入比對功能才不會錯誤
  3. 設定空陣列，等等可以藉由空陣列來裝取隨機陣列

寫入創造表格的函式

這裡我們將 comics 這個創建好的陣列進行操作，利用 map() sort() forEach() 創造新的陣列並且推入 DOM 顯示出來

  1. 寫入函式

  2. 將comics 陣列利用 map() 把陣列裡面的 value 丟到新的陣列中，並且寫入 sort 給予一個隨機亂數

  3. 接著寫入 sort()讓參數 a - b 可以達到從小排到大的效果就可以達到每次重整都會將清單打亂的效果 JS 將陣列 Array 重新排列的 sort()可以參考這篇文章

  4. 接著我們再一次寫入 map() 這次只抓取 value 的部分，得到一個新的陣列

  5. 寫入 forEach 參數設定為 (comicsname , index)

  6. 定義變數 listItem 為創造一個 li ，接著寫入 setAttribute 給予這個 li 屬性 ('data-index' , index)

  7. 接著寫入每個 listItem 的內容，前面為編號帶入 index+1 因為陣列從0開始，然後帶入comicsname把每個漫畫名稱丟進去，這邊的 draggable="true" 是為了讓整個 <div> 變成可以拖曳的，非常重要！

  8. 將這個跑完的 listItem 內容丟進去空陣列 listItems 裡面

  9. 把跑完的 listItem 丟進 draggable_list 當中就可以顯示出來了

  10. 寫入滑鼠拖曳事件

滑鼠拖曳事件

這裡本來有嘗試使用　JQuery 來寫，後來一直有 bug ，所以改回 JS ， 查一下發現滑鼠拖曳事件是在 JQuery UI 才有，但我沒使用過，之後會再來研究研究

  1. 這裡定義兩個變數，一個是選擇全部的.draggable 與選擇全部的 .draggable-list li' 我們等等是要對 .draggable-list li' 進行拖曳
  2. 我們先讓每個 .draggable 去跑一次拖曳事件 draggable.addEventListener('dragstart',dragStart); (dragstart是事件開始的寫法，跟播放音樂很像，都要先寫入 play())
  3. 接著讓每個 .draggable-list li' 都取跑”四個”事件，分別是「放下」「進入」「離開」「在目標時」事件
  4. 寫入開始事件函式 dragStart()，使用 closest()讓我們找到最接近拖曳物件的<li>的 data-index位置
  5. 寫入放下事件函式 dragDrop() ,得到放下時的data-index位置
  6. 寫入交換事件(等等寫入)
  7. 寫入進入事件函式 dragEnter() 當拖曳的物件經過其他物件時，該物件會加入一個 over的 class 會變更 </li>的背景顏色，增加互動
  8. 寫入進入再目標時的事件函式dragOver(e) 並且關掉預設 e.preventDefault(); ，因為不關掉的話，會干擾到 drop事件
  9. 寫入離開事件函式 dragLeave()當離開物件時，我們刪除它新增的 over class

寫入交換函式 swapItem

這邊寫入一個 swapItem 函式，它可以在放下(drop)時觸發，讓拖曳與放下目標進行更換

  1. 寫入函式，寫入兩個參數(拖曳的item , 放下的item)
  2. 定義 itemOne為陣列裡備拖曳的('.draggable');
  3. 定義 itemTwo為陣列裡備拖曳的('.draggable');
  4. 接著將兩者都互相插入達成交換的效果

寫入滑鼠點擊事件 check 列表功能

這邊寫入一個確認列表順序的功能，當列表選擇好順序後按下按鈕，會去匹配原本的 comics 陣列，看順序是否正確

  1. 寫入事件
  2. 讓我們亂序的 listItems 陣列去跑 forEach() ，讓陣列中的每個數據跑一次函式
  3. 定義變數comicsName，抓取每個陣列的數據的內文 (innerText)，這邊有使用trim()避免空白
  4. 寫入 if 判斷，若 listItems陣列當中的數據，不等於comics陣列裡面的數據時，給予 class

綠色為位置正確，紅色為位置錯誤
