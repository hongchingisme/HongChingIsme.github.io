# Javascript 作品

實作一個打字遊戲

作品

原始碼(有原始 JS 的 code)

本實作想實現的功能:

  1. 頁面會顯示亂數的文字
  2. 玩家輸入成功後會累積分數並且給予增加時間的獎勵
  3. 左下角的齒輪按下去之後，上面會顯示難度設定的選項
  4. 選擇不同難度，時間獎勵不同
  5. 遊戲結束之後，會顯示 Game over 的畫面，讓玩家 reload

實作內容

HTML 部分

這部分比較沒有複雜的地方，但還是講一下步驟

  • 完整的程式碼可從最上面連結觀看
  1. 寫入一個按鈕，加入設定的 icon
  2. 寫入 select ，裡面的選項為不同難度
  3. 寫入遊戲 container 裡面包住標題、顯示的文字、時間、分數、輸入框
  4. 寫入 game over ，但沒內容，之後會在 CSS . JS 進行設定

成果:

CSS 部分

CSS 部分比較要提的是動畫的手法，雖然之前有做過類似的，但感覺實戰會很常用到，所以特別拿出來筆記一

  • 完整的程式碼可從最上面連結觀看

成品:

這裡是將我們的難度選擇部分讓他可以在按鈕時顯示與隱藏上面的選項條，增加遊戲互動。

  1. 先寫入初始位置transform: translateY(0)
  2. 寫入增加 class 後的位置 transform: translateY(-100%)
  3. 寫入動畫，讓 transform 有 ease-in-out 的效果

Jquery 部分

抓取 DOM

寫入一個 文字的陣列，把要產生的文字寫進去

寫入這個陣列，是為了讓我們等等可以寫入一個抓取隨機陣列裡的文字時候用的，而網路上應該也有提供類似的 API ，要不然只有這幾個重複率應該很高。

設定一些數值的初始值

  1. 定義一個變數為隨機文字，等等會寫函式來操作
  2. 將初始分數歸零
  3. 將初始時間調成10

寫入隨機產生文字功能

  1. 寫入函式
  2. return 陣列裡面的亂數(這邊透過 floor 去取最跟最小整數，然後乘陣列長度)
  3. 將產生出來的文字顯示出來

寫入文字輸入監聽事件

  1. 當我們輸入的值等於隨機產生的文字時，就在更新一次文字，並讓分數刷新
  2. 分數刷新函式，讓分數自增，並讓新的分數寫入

寫入時間倒數與遊戲結束功能

  1. 一開始，我們先定義變數timeInterval = setInterval(updataTime , 1000); 使用setInterval將時間循環倒數(一啟動之後就會在固定的間隔時間不斷執行)，讓時間每一秒執行

  2. 接著寫入函式，讓他每一秒行一次自減

  3. 將新的時間寫入 HTML

  4. 寫入 if ，當時間歸零的時候，我們寫入clearInterval()來讓時間停止

  5. 時間到後呼叫 gameover 函式

  6. 寫入 gameover 函式

  7. 將原本沒內容的 endgame container 填入內容

  8. 讓它的 CSS 當中的 display 顯示出來為 flex

寫入設定按鈕與難度設置

  1. 當按下設定按鈕時，加入 hide 的 class 觸發 CSS 達成關閉與顯示效果
  1. 寫入 focus() 讓使用者進入頁面就自動從 input 開始
  2. 我們定義difficulty的值，讓它的值為瀏覽器儲存的值，若沒有值，就讓它為’Medium’
  3. 讓顯示的選擇器上的 value 為difficulty的值
  4. 寫入 change 監聽事件，當我們更換選擇器裡的選項時觸發
  5. 定義difficulty = difficultySelect.val();
  6. 將選擇的值保存在瀏覽器
  7. 選擇後讓網頁更新，遊戲重新開始

不同難度會有不同的時間獎勵

基於難度設定，給予玩家輸入正確的獎勵，當難度最高時間獎勵就越少，我們透過在剛剛的輸入監聽事件當中寫入，當玩家成功的輸入正確後，會依照難度給予不同的時間。
